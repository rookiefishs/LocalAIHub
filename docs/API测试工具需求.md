# API 测试工具需求文档

## 概述

本文档描述 LocalAIHub API 测试工具功能的需求，用于在管理界面直接测试 API 调用。

## 功能列表

### 1. 测试能力

- 选择已创建的 API Key 进行测试
- 选择虚拟模型
- 自定义请求参数
- 支持流式输出
- 显示完整响应信息

### 2. 测试场景

- Chat Completion（文字对话）
- 查看请求延迟、Token 消耗
- 验证路由是否正常工作

## 后端 API 设计

### 测试工具

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | /admin/api/v1/tools/test-request | 模拟 API 请求测试 |

### API 详情

#### POST /admin/api/v1/tools/test-request

**请求头**

```
Authorization: Bearer <API_KEY>
Content-Type: application/json
```

**请求体**

```json
{
    "model": "gpt-4o",
    "messages": [
        {"role": "user", "content": "你好，请介绍一下自己"}
    ],
    "temperature": 0.7,
    "max_tokens": 500,
    "stream": false
}
```

**响应示例**

```json
{
    "code": 0,
    "data": {
        "success": true,
        "model": "gpt-4o",
        "latency_ms": 1523,
        "prompt_tokens": 20,
        "completion_tokens": 45,
        "total_tokens": 65,
        "response": {
            "id": "chatcmpl-xxx",
            "object": "chat.completion",
            "created": 1704067200,
            "model": "gpt-4o",
            "choices": [
                {
                    "index": 0,
                    "message": {
                        "role": "assistant",
                        "content": "你好！我是..."
                    },
                    "finish_reason": "stop"
                }
            ],
            "usage": {
                "prompt_tokens": 20,
                "completion_tokens": 45,
                "total_tokens": 65
            }
        }
    }
}
```

### 流式响应

当 `stream: true` 时，后端返回 Server-Sent Events：

```
data: {"choices":[{"delta":{"content":"你"}}]}
data: {"choices":[{"delta":{"content":"好"}}]}
data: {"choices":[{"delta":{"content":"！"}}]}
data: [DONE]
```

### 3. 组件说明

1. **请求配置区**
   - API Key 下拉（从已创建的 Key 中选择）
   - 模型下拉（从虚拟模型中选择）
   - 请求体 JSON 编辑器（带语法高亮）

2. **响应展示区**
   - 状态码、耗时、Token 消耗
   - 响应体展示（JSON 格式化）
   - 流式输出时实时显示

### 4. 功能特点

- 请求体模板快速填充
- JSON 语法检查
- 响应时间、Token 消耗统计
- 请求历史记录（可选）
