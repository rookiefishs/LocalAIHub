# 配置导入导出需求文档

## 概述

本文档描述 LocalAIHub 配置导入导出功能的需求，实现一键导出/导入全部配置。

## 功能列表

### 1. 导出内容

导出配置包含以下模块：

| 模块 | 描述 |
|------|------|
| providers | 上游服务商配置 |
| provider_keys | 上游 API Key 配置 |
| virtual_models | 虚拟模型配置 |
| bindings | 模型绑定配置 |
| api_clients | 客户端 API Key 配置 |
| system_config | 系统配置 |

### 2. 导出文件格式

```json
{
    "version": "1.0",
    "exported_at": "2024-01-01T00:00:00Z",
    "exported_by": "admin",
    "providers": [
        {
            "id": 1,
            "name": "OpenAI",
            "provider_type": "openai",
            "service_type": "official",
            "base_url": "https://api.openai.com",
            "auth_type": "bearer",
            "timeout_ms": 60000,
            "enabled": true,
            "remark": ""
        }
    ],
    "provider_keys": [
        {
            "provider_id": 1,
            "key_masked": "sk-abc***xyz",
            "secret_encrypted": "encrypted_value",
            "priority": 1,
            "remark": ""
        }
    ],
    "virtual_models": [
        {
            "id": 1,
            "model_code": "gpt-4o",
            "display_name": "GPT-4o",
            "protocol_family": "openai",

            "status": "active",
            "sort_order": 10,
            "description": ""
        }
    ],
    "bindings": [
        {
            "virtual_model_id": 1,
            "provider_id": 1,

        }

        {
            "name": "测试Key",

    "system_config": {
        "gateway.default_timeout_ms": 60000,
        "circuit_breaker.failure_threshold": 5
    }
}
```

### 3. 导入逻辑

导入时执行以下操作：

1. 解析 JSON 文件，验证版本和格式
2. 校验必填字段
3. 如果配置已存在，选择覆盖或跳过
4. 批量创建/更新配置
5. 返回导入结果报告

### 4. 导入选项

| 选项 | 描述 |
|------|------|
| overwrite_existing | 覆盖已存在的配置，默认 true |
| skip_invalid | 跳过无效记录，默认 true |
| dry_run | 试运行，只返回预览不实际导入 |

## 后端 API 设计

### 配置管理

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /admin/api/v1/config/export | 导出全部配置 |
| POST | /admin/api/v1/config/import | 导入配置 |

### API 详情

#### GET /admin/api/v1/config/export

**请求参数**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| modules | string | 否 | 导出的模块，用逗号分隔，默认全部 |

**响应**

返回 JSON 文件下载。

#### POST /admin/api/v1/config/import

**请求头**

```
Content-Type: application/json
```

**请求体**

```json
{
    "config": { /* JSON 配置内容 */ },
    "options": {
        "overwrite_existing": true,
        "skip_invalid": true,
        "dry_run": false
    }
}
```

**响应示例**

```json
{
    "code": 0,
    "data": {
        "dry_run": false,
        "summary": {
            "providers": { "created": 2, "updated": 1, "skipped": 0 },
            "virtual_models": { "created": 3, "updated": 0, "skipped": 0 },
            "bindings": { "created": 5, "updated": 0, "skipped": 0 },
            "api_clients": { "created": 1, "updated": 0, "skipped": 0 }
        },
        "errors": []
    }
}
```

### 1. 入口
