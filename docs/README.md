## 当前功能状态

| 模块 | 当前状态 | 说明 |
|------|----------|------|
| 仪表盘 | 已上线 | 支持请求趋势、Token 趋势、模型分布、API Key 维度统计 |
| 上游管理 | 已上线 | 支持 Provider、Provider Key、连接测试、排序 |
| 虚拟模型 | 已上线 | 支持多协议模型、上游绑定、测试、拖拽排序 |
| 路由管理 | 已上线 | 支持手动切换、解锁、故障转移 |
| API Key 管理 | 已上线 | 支持权限、有效期、配额设置、使用示例 |
| API Key 配额管理 | 已上线 | 支持每日/每月请求与 Token 限制，超限自动禁用 |
| API 测试工具 | 已上线 | 支持选择 Key/模型并直接发请求测试 |
| 配置导入导出 | 已上线 | 支持全量或按模块导出、导入和试运行 |
| 操作审计日志 | 已上线 | 支持列表、详情、关键词筛选、CSV 导出 |
| 请求日志 | 已上线 | 支持筛选、详情查看 |
| Gemini 协议代理 | 已上线 | 已接入 `generateContent` / `streamGenerateContent` |
| 告警通知系统 | 需求完成，未开发 | 仅保留需求文档，代码尚未落地 |

## 已知边界

- Gemini 当前走生成接口转发，管理侧通过模型协议配置接入。
- 告警通知系统仍处于需求阶段，暂未看到后端/前端实现。
- 旧的“高级统计分析”模块已并入当前仪表盘，单独模块不再继续维护。
